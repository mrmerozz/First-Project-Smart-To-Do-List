# Smart To-Do List

A clean, responsive to-do app I built - add tasks, give them a priority, and watch your progress as you check things off. It saves everything in the browser so your tasks are still there when you come back, and it has a light and dark theme.

This was one of my first web projects, made with plain HTML, CSS, and JavaScript (no frameworks).

## What it does

- Add tasks and tag them as urgent, normal, or chill
- Check tasks off as done, or clear all the finished ones at once
- A progress bar and quick stats (total / done / left) that update as you go
- Light and dark theme toggle, plus a time-based greeting at the top
- Small motivational messages and a GIF that change as you make progress
- Saves your tasks with localStorage, so nothing is lost on refresh

## Running it

It's a static site, so you can just open `index.html` in your browser. It's also live on GitHub Pages:
